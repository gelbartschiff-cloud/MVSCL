## 📁 项目目录结构

本项目的主要代码结构如下：

```text
MVSCL-master/
├── custom_optimizers/ # 自定义优化器代码
├── dataloaders/       # 数据集加载与预处理模块
├── models/            # 存放训练好的模型权重文件 (例如：last-Res_Unet.pth)
├── networks/          # 核心网络结构定义
├── utils/             # 各种辅助工具函数和脚本
├── config.py          # 项目全局配置文件 (超参数、路径等)
├── run.sh             # 自动化运行脚本
└── TTA.py             # 测试时数据增强 (Test Time Augmentation) 脚本
